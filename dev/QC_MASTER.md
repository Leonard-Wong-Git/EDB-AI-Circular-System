# EDB 通告智能分析系統 — QC Master 規範
**完成版本：v3.0.46**
**制定日期：2026-04-17**
**適用環境：** GitHub Pages `https://leonard-wong-git.github.io/EDB-AI-Circular-System/`

> 此文件為系統正式完成版的 QC 黃金標準。每次日後維護或升級後，須對照此清單重新驗收。
> 驗收結果記錄於底部「驗收記錄」表。
> 凡標記 ✅ 的項目為已知通過；⬜ 為每次驗收需重新確認的項目。

---

## 驗收規則

1. **全通過定義**：所有 ⬜ 項目均得到「符合預期結果」
2. **阻塞級別**：A、B、C、J 類中任何一項失敗 = 不可標示為通過
3. **非阻塞級別**：K（響應式）、H（設定）失敗可記錄並追蹤，不阻塞
4. **回歸原則**：每次版本升級後，A+B+C+J 類必須完整重測
5. **快速驗收**（日常更新後）：最低執行 A1–A3、B6、C2、J6 六項

## 已知限制（不影響正式完成狀態）

1. EDB 舊年度通告 PDF 連結可能因 EDB 伺服器路徑變更而失效（以教育局官網為準）
