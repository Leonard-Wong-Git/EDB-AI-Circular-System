## 0. 架構原則（不變）

- 單一 HTML 檔案（edb-dashboard.html），內嵌所有 CSS + JS
- 無框架、無建構工具（Vanilla HTML/CSS/JS）
- 外部依賴只允許 CDN（SheetJS、JSZip）
- `circulars.json` 與 HTML 同目錄，頁面載入時 fetch
- 所有用戶設定存入 `localStorage`，頁面載入時自動還原

## 2. Header

### 2.1 角色選擇器
- Dropdown，選項：校長 / 副校長 / 科主任 / 教師 / 行政主任 / 供應商
- 選擇後：
  - 整頁通告依角色過濾 `role_relevance`
  - 供應商 → 顯示「🏢供應商」Tab，自動切換至供應商 Tab
  - 非供應商 → 隱藏「🏢供應商」Tab
  - 設定存入 `localStorage: edb_role`

### 2.2 搜尋欄
- 輸入即時過濾（debounce 300ms）
- 搜尋範圍：通告號、標題、摘要
- 輸入時顯示自動建議下拉（最多 8 條）
- 手機版：收起為 🔍 圖示，點擊展開全寬搜尋欄

### 2.3 語言切換
- 中 / EN 切換按鈕
- 介面文字切換，通告內容維持原語言
- 設定存入 `localStorage: edb_lang`

| 學校類別 | 小學 / 中學 / 特殊學校 / 幼稚園 / 全部 |

## 5. 通告總覽 Tab（📊）

### 5.1 視圖切換

右上角按鈕切換：**卡片視圖** ↔ **列表視圖**（設定存 localStorage）

#### 卡片視圖

```
┌──────────────────────────────────────┐
│ 🔴 需關注  [已跟進 ✓]  [⭐收藏]       │  ← 高影響通告才有紅色頂條
│ EDBCM030/2026                        │
│ 優質教育基金特別撥款計劃              │
│ 2026-03-01 | 💰HK$800K（可申請）      │

│ 🔵 校長 高  🔵 副校長 高             │
└──────────────────────────────────────┘
```

**「需關注」置頂區塊：** 頁面頂部獨立顯示所有 `impact: high + mandatory` 通告，以紅色橫條區隔。

#### 列表視圖（Notion 風格）

| 通告號 | 標題 | 日期 | 影響 | 行動 | 截止 | 金額 | 狀態 |
|--------|------|------|------|------|------|------|------|
| CM030/2026 | 優質教育基金... | 2026-03-01 | 🔴 | 必須 | 04-30 | HK$800K | 待跟進 |

- 點擊欄位標題可排序（日期 / 影響級別 / 截止日期 / 金額）
- 支援欄位顯示/隱藏（右上角 ⚙️ 欄位選擇器）

### 5.2 通告狀態追蹤（B5）

每條通告右上角 3 態切換按鈕：
- ⬜ 未讀（預設）
- 👁️ 已讀
- ✅ 已跟進（對應 A3 「手動 tick 已跟進」）

狀態存入 `localStorage: edb_status_{circularId}`

### 5.3 匯出（A8 擴充）

匯出選項（下拉選單）：
- **Excel 匯出**（現有功能保留）：通告列表匯出為 .xlsx
- **文件包下載**（新增）：選取一條或多條通告後，打包下載 ZIP，內含：
  - 該通告的 JSON 資料
  - 分析摘要 PDF（由前端生成）
  - 原始 PDF 連結清單（txt 格式，供用戶自行下載）

## 7. 資源申請 Tab（💰）

新增分頁，只對學校角色顯示（非供應商）。

### 7.1 資源申請列表

過濾出含撥款/申請資訊的通告：

| 通告號 | 資助名稱 | 金額/資源 | 類型 | 申請截止 | 狀態 | 適用角色 |
|--------|---------|-----------|------|----------|------|----------|
| CM030 | 優質教育基金特別撥款 | HK$800K 💰可申請 | 直接撥款 | 04-30 | 待申請 | 校長 |

狀態欄可手動更新（存 localStorage）：
- ⬜ 待申請
- 📝 申請中

- ➖ 不適用

### 7.3 文件包下載（A8）

選取申請項目後，下載 ZIP 包含：
- 通告摘要（PDF）
- 申請截止提醒（ics 日曆格式）

## 10. 詳情 Panel（右側 Overlay / 手機 Bottom Sheet）

### 10.1 5 個內部 Tab（保留原有，擴充）

| Tab | 內容 |
|-----|------|
| 📝 摘要 | LLM 摘要、影響評估、金額標示（💰/📦）|
| ✅ 行動 | structured_actions，按截止日期排序，加入狀態 tick |
| 📅 截止 | structured_deadlines，顏色倒數提示（7天橙/3天紅）|
| 👥 角色 | role_relevance accordion，當前角色置頂高亮 |
| 🔄 版本比較 | diff 並列顯示（B1：左舊版 / 右新版，顏色高亮變更）|

### 10.2 截止倒數提示（B2）

截止日期顯示規則：

| 距截止 | 顏色 | 附加 |
|--------|------|------|
| > 14 天 | 灰色 | 無 |
| 7–14 天 | 橙色 | ⚠️ |
| 3–6 天 | 深橙 | ⚠️ 即將截止 |
| < 3 天 | 紅色閃爍 | 🔴 緊急 |
| 已截止 | 灰色刪除線 | ✕ 已截止 |

**重要：** 截止類型由 LLM 提供，分為：
- `apply_deadline`：申請截止（必須行動）
- `submission_deadline`：提交報告截止
- `awareness_deadline`：需知悉/生效日期（非必須行動）

截止提示會配合角色顯示（例如申請截止只對負責申請的角色顯示緊急警示）。

### 10.3 列印 / PDF 匯出（B6）

詳情 Panel 右上角「列印」按鈕：
- 觸發 `window.print()`，CSS `@media print` 隱藏導航/Panel 框架
- 只印通告內容：摘要、行動、截止日期、角色分析

## 15. circulars.json Schema 擴充（後端對應）

前端新功能需要後端 LLM 提供以下額外欄位：

```json
{
  "llm": {
    "grant_info": {
      "type": "applicable | resource | none",
      "amount_hkd": 800000,
      "amount_label": "HK$800,000",
      "resource_value_hkd": null,
      "note": "學校可直接申請，上限 HK$80萬"
    },
    "structured_deadlines": [
      {
        "date": "2026-04-30",
        "type": "apply_deadline | submission_deadline | awareness_deadline",
        "role": ["principal", "eo_admin"],
        "label": "申請截止"
      }
    ]
  }
}
```
